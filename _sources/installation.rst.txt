Installation
============

- **Clone the repository**:

  .. code-block:: bash

      git clone https://github.com/bhatishan2003/weathora.git
      cd weathora

- **Install the package**:

  .. code-block:: bash

      pip install .

- **For development (editable mode)**:

  .. code-block:: bash

      pip install -e .

- **For setting API key**:

  In **Bash**:

  .. code-block:: bash

      export OPENWEATHER_API_KEY="YOUR_SECRET_KEY"

  In **PowerShell**:

  .. code-block:: powershell

      $Env:OPENWEATHER_API_KEY = "YOUR_SECRET_KEY"
